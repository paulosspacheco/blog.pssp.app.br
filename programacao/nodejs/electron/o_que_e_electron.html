  1. Introdução

    1. Objetivo:

      1. Electron é um framework de software de código aberto desenvolvido e mantido pelo GitHub .Ele permite o desenvolvimento de aplicações desktop GUI usando tecnologias web: ele combina o motor de renderização Chromium e o tempo de execução Node.js. Electron é o framework GUI principal por trás de vários projetos de código aberto, incluindo Atom , GitHub Desktop , Light Table , Visual Studio Code , Ever note e WordPress Desktop.
