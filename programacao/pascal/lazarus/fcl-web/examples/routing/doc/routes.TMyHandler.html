    • A classe TMyHandler implemnta o método TRouteObject.HandleRequest()* para executar a procedure RequestToResponse('Route object',ARequest,AResponse).

Métodos

procedure HandleRequest(ARequest: TRequest; aResponse: TResponse); override;

O método HandleRequest é um método abstrato que obrigatóriomente precisa ser anulado para que se possa executar o que se deseja.

  • DESCRIÇÃO:

    • Neste exemplo é executado a procedure RequestToResponse('Route object',ARequest,AResponse);

  • PARÂMETROS

    • ARequest

      • Contém informações vinda do Browser;

    • aResponse

      • Contém informações enviadas para o Browser;

    • EXEMPLO DE USO:

      • Registro do objeto TMyHandler:

        • HTTPRouter.RegisterRoute('/routed/object',rmall,TMyHandler);

      • Chamada do método TMyHandler.HandleRequest()

        • http://localhost:8080/routed/object/
