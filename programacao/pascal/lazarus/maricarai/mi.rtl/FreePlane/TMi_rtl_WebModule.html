navigation actions
CmlocateRequest ✅ 
       - Recebe a chave enviada pelo cliente;
        - Checa se o evento onLocate foi iniciado;
          - Se sim:     
        - Executa onLocate com parâmetros recibos,
          - Se não:
      - Executa método locate com os parâmetros recebidos;
        - Checa se o registro foi localizado:
          - Se sim:
            - Retorna para o cliente todos os campos TDmxScroller_form.JSONObject.
        - Se não:
          - Retorna o json: {"error":"Usuário não encontrado"}

Comandos usados pela aplicações lcl

CmGoEof ✅

Posiciona  no útimo registro.

LCL TActionList ✅

Comandos usados pela aplicações lcl

CmNewRecord ✅

Posiciona o curso no estado, inicializa com os valores padrões informados no método getTemplate.

CmUpdateRecord ✅

Caso o estado atual do registro se novoregistro então execute o método DoAddRec, caso contrário executa o método doUpdateRecord.

CmDeleteRecord ✅

Exclui o registro selecionado.

CmCancel ✅

Cancela a edição dos dados digitados até o mento e não gravados.

CmRefresh ✅

Ler novamente o  registro corrente do arquivo a atuliza a interface visual;.
