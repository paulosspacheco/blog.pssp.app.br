  1. Resumo do conteúdo:

    1. Este projeto está na versão Alpha, por isso não recomendo seu uso, porque o mesmo pode ser mudado antes do lançamento da versão estável.

  2. Introdução

    1. Objetivo:

      1. O pacote RTL contém tudo que preciso para:

        1. Acesso ao sistema operacional das plataformas que o free pascal reconhece.
        2. Acesso ao framework gráfico que o free pascal reconhece;
        3. Acesso aos bancos de dados que o free pascal reconhece.

      2. [🔙]

    2. Pre-requisitos:

      1. Linguagem Pascal versão Delphi.

      2. Conhecimento das units do free pascal.

      3. [🔙]

    3. Benefícios:

      1. O pacote mi.rtl.objectss tem todas as units de acesso a arquivos, banco de dados, gráficos declarando apenas a unit mi.rtl.objectss.

      2. [🔙]

    4. Desvantagens.

      1. O menor programa conterá um tamanho maior que se declarar a unit original apenas.

      2. [🔙]
