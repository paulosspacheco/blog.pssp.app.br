Instalação do Serviço NFS

  1. Para instalar o servidor NFS instale o pacote nfs-kernel-server.

    
   sudo apt update

   # Este app deve ser instalado na máquina que vai compartilhar os arquivos.
   sudo apt install nfs-kernel-server

  2. Para instalar o cliente NFS na máquina cliente instale o pacote nfs-common.

    
   sudo apt update

   # Este app deve ser instalado na máquina cliente
   sudo apt install nfs-common
