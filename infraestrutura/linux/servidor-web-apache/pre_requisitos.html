Pre-requisito para instalar o servidor web apache2

  1. Sistema operacional Linux ou windows instalado;

  2. Comunicação com a internet;

  3. Direitos de root ao sistema operacional;

  4. Noções básicas de redes de computadores;

  5. Noções básicas de Linux em linha de comando (Gerenciamento de diretórios e arquivos);

  6. Saber usar um editor de texto qualquer;

  7. Espaço em disco necessário:

    1. Para os programas:
      • 20 MG
    2. Para os sites:
      • Depende do tamanho dos sites.

  8. Para que o site fica disponível ao público é necessário instalar e configurar o servidor de DNS bind9;

  9. Opcional:

    1. Abrir a porta tcp:80 e permitir tráfego http, caso queira tráfego HTTPS a porta tcp:443 deve ser aberta. Se quiser abrir portas diferentes, configure o firewall, depois configure o Apache para usar o firewall.
